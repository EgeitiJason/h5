# h5-it-service

IT Service automation project using PowerShell.

## Folder Structure

| Folder | Description |
|---|---|
| `scripts/ActiveDirectory/` | User provisioning, group management, password resets, audits |
| `scripts/SystemAdmin/` | Disk monitoring, service restarts, backups |
| `scripts/Network/` | Network diagnostics, firewall rules, DNS |
| `scripts/Helpdesk/` | Ticket automation, asset inventory, SLA reporting |
| `modules/` | Shared reusable PowerShell modules (.psm1) |
| `config/` | Configuration files (JSON/XML settings) |
| `docs/` | Project documentation and guides |
| `logs/` | Script execution logs (gitignored) |
| `reports/` | Generated reports (gitignored) |
| `tests/` | Pester tests |
